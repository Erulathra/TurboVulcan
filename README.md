# Turbo Vulcan Engine

There will be a Game engine written in C++ using Vulcan.

## Dependencies (Fedora)

### Libraries

* `spdlog-devel`
* `sdl3-devel`
* `glm-devel`
* `vulkan-headers`
* `vulkan-tools`
* `vulkan-loader-devel`
* `vulkan-validation-layers`
* `VulkanMemoryAllocator-devel`

### Tools

* `glslc`
